# Global EV Sales Analysis using Power BI

## Project Overview

This project analyzes global Electric Vehicle (EV) sales using Power BI. It focuses on transforming raw data into interactive dashboards to uncover trends, regional performance, and market growth.

---

## Objectives

* Analyze global EV sales data
* Identify growth trends and patterns
* Compare regional performance
* Track KPIs like total sales and market share
* Support data-driven decision-making

---

## Data Preparation

* Imported dataset into Power BI
* Removed duplicates and handled missing values
* Converted data into proper formats
* Created DAX measures (Total Sales, Growth Rate, Market Share)

---

## Dashboard Features

* Trend Analysis: EV growth over time
* Regional Analysis: Country-wise performance comparison
* Model Analysis: Distribution across vehicle types
* KPIs: Sales, growth rate, and market share

---

## Key Insights

* Significant EV growth after 2020
* China leads global EV sales
* Emerging markets show rapid expansion
* BEVs dominate EV adoption
* Policies impact EV growth trends

---

## Challenges

* Data integration from multiple sources
* Handling missing data
* Creating advanced DAX calculations

---

## Conclusion

This project demonstrates how Power BI can be used to analyze large datasets, build interactive dashboards, and generate insights to support data-driven decisions in the EV industry.

---
